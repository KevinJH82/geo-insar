项目修复建议

  • HyP3 端点不通:检查 OpenVPN 出口(参考 commons/network.py 的 resolve_proxies)
  • HyP3 授权缺失:登 https://hyp3-api.asf.alaska.edu/ui/ 用 Earthdata 账号授权 HyP3 应用
  • Earthdata 凭证缺失:复制 /opt/deepexplor-services/geo-downloader/credentials.yaml.example 到 config/credentials.yaml 并填写 nasa_earthdata 段
  • Python 依赖缺失:在 geo-insar 目录运行 pip install -r requirements.txt
  • 端口被占:修改 config.yaml 中的 port(默认 8084),或释放冲突进程

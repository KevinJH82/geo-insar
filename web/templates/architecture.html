InSAR 干涉对标准目录(每对一个目录) 

downloads/<aoi_name>/sentinel1_insar/<refdate>_<secdate>_<polarization>/
  ├── unwrapped_phase.tif    # 解缠相位(弧度)
  ├── coherence.tif          # 相干性(0–1)
  ├── los_displacement.tif   # LOS 形变(mm)
  ├── wrapped_phase.tif      # 干涉图(可选)
  ├── dem.tif                # DEM(可选,--include-dem)
  ├── water_mask.tif         # 水体掩膜(可选,--include-water-mask)
  ├── incidence_angle_map.tif# 入射角图(可选,--include-inc-map)
  ├── metadata.json          # 遵循 commons/insar_schema.json
  └── _raw/                  # HyP3 原始 ZIP 解压(供调试,不参与下游消费)

每个 AOI 下还有一个 stack_index.json 索引所有干涉对的统计概要,供下游快速查询。

跨子系统数据流 

geo-insar/  →  标准输出目录(本契约)
    │
    ├──→ geo-exploration : 定时扫描 → SlowVars 第 8 类因素 → 矿产预测图叠加
    │
    ├──→ geo-reporter    : insar_reader 读统计 → categories 新章节 → 报告 InSAR 章节
    │
    └──→ geo-analyser    : insar_broker 扫描 → insar_analysis / timeseries → API/前端
